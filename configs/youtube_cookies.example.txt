# YouTube cookies for streammon member checks
#
# yt-dlp's persistent-cookie instructions:
# https://github.com/yt-dlp/yt-dlp/wiki/extractors#exporting-youtube-cookies
#
# Export cookies in Netscape format, then replace the contents of this
# youtube_cookies.txt file with the exported cookie data.
#
# Use a private browsing/incognito window for the export flow described by
# yt-dlp. After exporting, close that window so YouTube does not rotate the
# exported session cookies.
#
# Keep youtube_cookies.txt private. It grants access to your signed-in YouTube
# account and must not be committed, uploaded, or shared.
#
# streammon uses this file for members-only stream discovery and downloads.
# By default, member downloads use livestream_dl with this cookie file because
# YouTube downloads with yt-dlp cookies can stall indefinitely.
#
